# 회원가입 문제 해결

회원가입이 안 되는 경우 다음을 확인하세요:

## 1. Prisma 클라이언트 재생성

스키마를 변경했으므로 Prisma 클라이언트를 재생성해야 합니다:

```powershell
cd server
npm run prisma:generate
```

## 2. 데이터베이스 마이그레이션

이메일 필드를 제거했으므로 마이그레이션이 필요합니다:

```powershell
npm run prisma:migrate
```

마이그레이션 이름: `remove_email`

## 3. 서버 재시작

코드 변경 후 서버를 재시작하세요:

```powershell
npm run dev
```

## 4. 브라우저 콘솔 확인

브라우저 개발자 도구(F12)를 열고 Console 탭에서 에러 메시지를 확인하세요.

## 5. 서버 로그 확인

서버 터미널에서 에러 메시지를 확인하세요. "Registration error:"로 시작하는 로그를 찾아보세요.

## 가능한 오류 원인

1. **데이터베이스 스키마 불일치**: 마이그레이션을 실행하지 않아서 email 필드가 아직 존재하는 경우
2. **Prisma 클라이언트 미갱신**: `prisma:generate`를 실행하지 않아서
3. **서버 미실행**: 백엔드 서버가 실행되지 않은 경우
4. **CORS 문제**: 프론트엔드와 백엔드가 다른 포트에서 실행 중이고 CORS 설정이 잘못된 경우
