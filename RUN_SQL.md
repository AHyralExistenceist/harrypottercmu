# 데이터베이스 스키마 실행 방법

## 빠른 실행 가이드

1. Supabase Dashboard 접속: https://supabase.com/dashboard/project/unnpnjxdqtrqvrcphmar
2. 왼쪽 메뉴에서 **SQL Editor** 클릭
3. **New query** 버튼 클릭
4. 아래 SQL 파일 내용을 복사해서 붙여넣기:
   - 파일 위치: `server/supabase-migration.sql`
   - 또는 아래 전체 SQL을 복사
5. **Run** 버튼 클릭 (또는 Ctrl+Enter)
6. 성공 메시지 확인
